รายการบัญชีล่าสุด

วัตถุดิบคลังสินค้า (ใกล้หมด/ต้องสั่งเพิ่ม)

วันที่ ประเภท รายการบัญชี หมวดหมู่ จำนวนเงิน จัดการ

ไม่พบข้อมูลรายการบัญชีในตัวกรองนี้

คลังและสต็อกวัตถุดิบอาหาร (Inventory)

จัดการวัตถุดิบนำเข้า บันทึกเพิ่มจำนวนสต็อก และตรวจเช็กยอดคงเหลือ

รหัส วัตถุดิบ หมวดหมู่ คงเหลือปัจจุบัน ราคาทุนต่อหน่วย การกระทำ

ไม่มีรายการวัตถุดิบในคลังขณะนี้ กดปุ่มด้านบนเพื่อเพิ่มสินค้าเข้าสต็อก

ควบคุมรายจ่ายคงที่และงบประมาณร้าน (Fixed Costs)

กำหนดเป้าหมายค่าใช้จ่ายคงที่ เช่น ค่าเช่า, เงินเดือนพนักงาน, ค่าน้ำไฟแก๊ส เพื่อไม่ให้ยอดใช้จ่ายบานปลาย

เชื่อมต่อระบบคลาวด์ & สำรองฐานข้อมูล

ผูกเชื่อมโยงข้อมูลหน้าเว็บเข้ากับ Google Sheets และ Google Apps Script

การตั้งค่า Google Sheets (Cloud API Connection)

ป้อนลิงก์ Web App URL ที่คุณสร้างขึ้นมาจาก Google Apps Script เพื่อเปิดใช้งานฟังก์ชันการซิงก์ข้อมูลอัจฉริยะร่วมกับ LINE Bot, สแกน Gmail และ Google Sheets คลาวด์อัตโนมัติ

ส่งออกข้อมูลบัญชีออฟไลน์ (Export JSON)

ดาวน์โหลดข้อมูลการทำงานปัจจุบันทั้งหมดลงเครื่องคอมพิวเตอร์ของคุณเพื่อเก็บเป็นไฟล์สำรองข้อมูลส่วนบุคคลได้ทันที

นำเข้าข้อมูลสำรอง (Import JSON)

เลือกไฟล์ข้อมูลที่เคยบันทึกไว้เพื่ออัปโหลดกลับเข้ามาเขียนทับและบันทึกทดแทนประวัติการเงินร้านในระบบปัจจุบัน

รีเซ็ตลบข้อมูลร้านค้าทั้งหมด (Wipe Data)

การดำเนินการนี้จะลบรายการบัญชี งบประมาณรายจ่ายคงที่ และสต็อกทั้งหมดออกจากบราวเซอร์ทันที
